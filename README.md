## 🌟 Overview

This is an **interactive farewell website** dedicated to our amazing teacher! Built with **HTML, CSS, and JavaScript**, it’s designed to make our send-off memorable, fun, and heartfelt.

The website includes:

- 🖼 **Photo Gallery** – Look back at all the fun and memorable moments.
- 📹 **Video Section** – Farewell clips to cherish the memories.
- 💌 **Teacher’s Message** – A special section where the teacher can leave a personal message.
- ✨ **Interactive Features** – Smooth animations, hover effects, and dynamic content.
